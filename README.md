# Zoluxiones - Reto Técnico

Proyecto frontend desarrollado con React + Vite utilizando Redux Toolkit, React Query y React Router.

---

## 🚀 Tecnologías utilizadas

- React 18
- Vite
- Redux Toolkit
- React Query
- React Router DOM
- Sass
- Swiper
- Vitest + Testing Library

---

## 📋 Requisitos previos

Antes de iniciar el proyecto, asegúrate de tener instalado:

- Node.js (recomendado: versión 18 o superior)
- npm o yarn

Puedes verificar tus versiones con:

```bash
node -v
npm -v
```

---

## ⚙️ Instalación

Clona el repositorio:

```bash
git clone https://github.com/randol514/zoluxiones-reto-tecnico
```

Ingresa al proyecto:

```bash
cd zoluxiones-reto-tecnico
```

Instala las dependencias:

```bash
npm install
```

---

## ▶️ Levantar el proyecto en desarrollo

Ejecuta el siguiente comando:

```bash
npm run dev
```

Luego abre en el navegador:

```bash
http://localhost:5173
```

---

## 🏗️ Build de producción

Para generar la versión optimizada del proyecto:

```bash
npm run build
```

Los archivos generados estarán en la carpeta:

```bash
dist/
```

---

## 👀 Vista previa del build

Para probar el build localmente:

```bash
npm run preview
```

---

## 🧪 Ejecutar tests

Correr los tests con:

```bash
npm run test
```

---

## 📦 Scripts disponibles

| Comando           | Descripción                      |
| ----------------- | -------------------------------- |
| `npm run dev`     | Inicia el servidor de desarrollo |
| `npm run build`   | Genera el build de producción    |
| `npm run preview` | Previsualiza el build            |
| `npm run test`    | Ejecuta las pruebas con Vitest   |
